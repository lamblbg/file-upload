## 安装  

### 前提条件  

- 确保你的开发环境已经安装了 Node.js
### 步骤  

1. 克隆项目到本地  
   ```bash  
   git clone git@github.com:lamblbg/file-upload.git

1. 进入项目目录

   ```bash
   cd file-upload
   ```

2. 安装依赖

   ```bash
   npm install
   ```

## 使用

### 基本用法

```bash
node .\src\app.js
```
